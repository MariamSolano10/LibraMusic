# LibraMusic

Aplicació web desenvolupada com a projecte de 2n de DAW. Plataforma de streaming de música de codi obert i lliure accés, centrada en la comunitat.

## Equip
- Product Owner: Mariam Solano
- Scrum Master: Joan Garcia
- Developers: Luisa Castellanos, Yangjia Jin, Ivan Ballespí

## Tecnologies
- Frontend: React / HTML CSS JS
- Backend: Node.js
- Base de dades: MongoDB
- Control de versions: Git + GitHub

## Metodologia
Aquest projecte es desenvolupa utilitzant SCRUM amb sprints de 2 setmanes.

## Instal·lació
git clone https://github.com/MariamSolano10/LibraMusic
npm install
npm run dev
